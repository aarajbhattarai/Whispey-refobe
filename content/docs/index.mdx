## Overview

Refobe is a professional voice analytics platform that helps you monitor, analyze, and gain insights from your AI voice agent conversations. Track performance metrics, analyze conversation quality, and optimize your voice agents with comprehensive analytics.

## Key Features

### Real-time Analytics
- Speech-to-Text (STT) metrics: Audio duration, processing time
- Large Language Model (LLM) metrics: Token usage, response time
- Text-to-Speech (TTS) metrics: Character count, audio duration
- Voice Activity Detection (VAD): Voice detection metrics
- Conversation flow and turn tracking

### Conversation Analytics
- Full transcript with timestamps
- Turn tracking for user and agent interactions
- Performance insights and response times
- Success metrics and call completion rates

### Dashboard Integration
View detailed analytics at the [Refobe Dashboard](https://whispey.xyz/):

- Call performance metrics
- Voice quality analysis
- Conversation flow patterns
- Usage statistics and cost tracking
- Business metrics and customer satisfaction

## Getting Started

1. **Install Refobe**: `pip install whispey` or install from [PyPI](https://pypi.org/project/Refobe/)
2. **Get Credentials**: Sign up at [Refobe Dashboard](https://whispey.xyz/)
3. **Configure Environment**: Set up your API key and agent ID
4. **Integrate**: Add Refobe to your LiveKit voice agent
5. **Analyze**: View insights in the dashboard
